생성형 AI를 활용하여 웹사이트 제작하기

ChatGPT를 활용하여 나만의 웹사이트를 제작해보는 예제입니다.
html, css, javascript를 사용합니다!

들어가기 앞서

많은 사람들이 코딩을 '어렵고 복잡한 무언가'로 여기며 시작조차 망설이곤 합니다. 하지만 이제는 생성형 인공지능(AI)의 도움으로 누구나 쉽게 여러 생소한 기술들을 시도해볼 수 있는 시대가 되었습니다! 생성형 AI를 활용해 간단한 웹사이트를 제작해보면서, '내가 할 수 있는 일'을 넓혀보는 경험을 함께 하셨으면 좋겠습니다.

🌱 1단계: 아이디어 구현 방법 제공 + 예제 코드

막연히 웹사이트를 만들어보겠다고 던져주기만 해도 챗GPT가 웹 개발의 기본 개념을 간단히 설명해주고, 바로 실행해볼 수 있는 예제 코드도 함께 제공해줍니다. 초보자도 이해할 수 있도록 설명해주고, 메모장을 이용해 직접 실습해보며 제작이 가능합니다.

🔧 2단계: 예제 코드 실행 및 코드 분석

챗GPT가 준 예제코드를 메모장을 이용해 실행해본 결과입니다! 생각보다 예쁘게 잘 만들어주네요🤔 마침 저희집 강아지들 자랑도 할 겸 이제 이 예제코드에서 이미지를 추가해보는 등 사이트를 수정하는 방법을 물어보겠습니다!

🧰 3단계: 예제 코드 발전 + 템플릿 추천

이 예제 코드를 발전시켜서 좀더 본격적인 우리집 강아지 자랑을 해보려고 하니, 사진 올리는 코드가 생각보다 번거롭고 그다지 예쁘지도 않더라고요😪 아예 예쁜 템플릿을 추천받고, 그 템플릿의 내용을 변경하는 방법을 물어봤습니다!

🧪 4단계: 템플릿 코드 수정 + 에러 해결

예쁜 템플릿이 있는 사이트를 추천해주네요!! 마음에 드는 걸 하나 골라서 입맛대로 코드를 수정해봤습니다. 배경 색을 수정하고, 글자 색도 수정하고, 이미지도 강아지 사진들로 채우는 방법을 물어보면서 코드를 수정했는데, 사진을 바꾸니까 사진 속 글자가 잘 안보여서 사진 속 글자만 하얀색으로 바꿔보고.. 생각보다 오래걸리네요

마지막으로

처음엔 막연히 '업무 관련 정리 페이지를 만들면, 내부망이 아니어도 볼 수 있고 인수인계할 때도 편하겠다!'는 마음으로, 업무 관련 정보를 공유할 수 있는 웹사이트 제작을 생각했습니다. 하지만, 업무 관련 사항을 공개 사이트에 함부로 게시했다가 발생할 수 있는 여러 문제들이 우려되어 현재는 개인 페이지로 노선을 변경하게 되었습니다. 누군가는 더 좋은 아이디어로 멋지게 자신만의 웹페이지를 만들 수 있으리라 기대해봅니다!
